# 科研方向咨询与导师推荐

**⚠️ 必须先运行脚本查询校内教师，不能凭记忆或猜测给出老师信息。**

脚本路径：`$PI_SKILLS_ROOT/research-advisor/research-find.py`

---

## 三段式回复结构（必须按此顺序）

### 第一部分：方向简介（2-3句）
- 说明核心内涵、技术路线、现状
- 结合农业/农大特色（如涉及农业应用则着重说明）
- 语气积极但客观

### 第二部分：我校相关教师（必须通过脚本获取，不允许编造）

**Step 1：提取关键词（1-3个）**

| 用户说的方向 | 搜索关键词 |
|------------|-----------|
| 农业具身智能 | 视觉 机器人 具身 |
| 智慧农业/精准农业 | 精准 农业机械 传感 |
| 计算机视觉/机器视觉 | 视觉 图像 检测 |
| 自然语言处理/大模型 | 语言 文本 知识图谱 |
| 深度学习/神经网络 | 深度学习 神经网络 学习 |
| 机器人 | 机器人 控制 路径规划 |
| 物联网/嵌入式 | 嵌入式 传感器 物联网 |
| 数据科学/大数据 | 数据 挖掘 图谱 |

**Step 2：运行查询脚本**
```bash
python3 $PI_SKILLS_ROOT/research-advisor/research-find.py <关键词1> <关键词2> <关键词3>
```
例如：用户问"农业具身智能" →
```bash
python3 $PI_SKILLS_ROOT/research-advisor/research-find.py 视觉 机器人 具身
```

**Step 3：呈现脚本输出**（完整展示教师列表）

若脚本返回结果较少（<2人），尝试更宽泛的关键词再次查询：
```bash
python3 $PI_SKILLS_ROOT/research-advisor/research-find.py 智能 感知 控制
```

### 第三部分：行动建议（选出1-2位最匹配的教师）
从第二部分结果中选1-2位最匹配的，给出具体联系建议：
- 推荐切入方式：
  - **竞赛**：互联网+创新创业大赛、挑战杯、全国大学生电子设计竞赛、全国农业机器人大赛
  - **课程科研项目**：联系老师询问是否有开放的科研助理名额
  - **发邮件**：简要说明背景和兴趣，表达加入意愿
- 提供推荐老师的完整信息：姓名、职称、邮箱
- 鼓励大胆主动联系

## 完整示例（"想研究农业具身智能"）

**执行**：
```bash
python3 $PI_SKILLS_ROOT/research-advisor/research-find.py 视觉 机器人 具身
```

**输出模板**：
```
【方向简介】
农业具身智能将机器人感知、规划与执行能力融入农业作业场景（采摘、巡检、耕作），是当前人工智能与农业工程交叉的前沿方向。随着视觉大模型和边缘计算突破，该方向近两年在国内外快速升温，农业场景的规模化应用场景是全球竞争热点。

【我校相关教师】
（research-find.py 的完整输出）

【行动建议】
建议优先联系以上1-2位与您方向最匹配的老师：

推荐方式：
1. 以参加「全国农业机器人大赛」或「互联网+」竞赛为切入点，主动联系老师询问是否可以组队指导
2. 发邮件说明自己的背景（年级、方向兴趣、技能），询问是否有开放的科研项目名额
3. 也可直接到老师办公室预约拜访（信电楼所在楼层）

重点联系：
— [从结果中选1-2位] | [邮箱]
```

## 注意事项

- **严禁编造**任何教师姓名、邮箱、研究方向——必须来自脚本输出
- 若脚本输出"未找到相关教师"，说明关键词太窄，换宽泛词再试
- bio 中研究方向较长时，只摘取最相关的关键词（3-5个），不全文复制
- 竞赛推荐要具体，结合方向选对口赛事
