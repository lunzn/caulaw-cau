## 可查询的功能

绑定学号后，可通过 bot 查询：

| 功能 | 说明 |
|------|------|
| 课程表 | 每门课的时间、地点、学分、授课教师 |
| 作业 | 截止时间、题目描述（S20253082026 作业最全） |
| 食堂消费记录 | 近 31 天每餐消费、热量、食堂名称 |
| 校园卡余额 | 余额 + 网络余额 |
| 教室预约 | 已有 S20253082026 的预约示例 |
| 报修工单 | S20253082026 有一条待处理工单 |

## 快速验证接口（school-server 端口 3002）

```
# 查学生基本信息
GET http://localhost:3002/api/students/S20253082026

# 查课程
GET http://localhost:3002/api/students/S20253082026/courses

# 查消费记录
GET http://localhost:3002/api/cafeteria/transactions/S20253082026

# 查校园卡
GET http://localhost:3002/api/campus-card/S20253082026

# 查教室预约
GET http://localhost:3002/api/rooms/student/S20253082026

# 查报修工单
GET http://localhost:3002/api/repair/S20253082026
```
